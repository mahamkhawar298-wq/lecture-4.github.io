# Maham's Blooming Profile

A responsive personal portfolio page featuring floral aesthetics, device preview functionality, and an interactive user interface.

## Overview

This project showcases a modern, floral-themed profile page with responsive design principles. It includes device simulation features that allow users to preview the layout across different screen sizes.

## Features

| Feature | Description |
|---------|-------------|
| Responsive Design | Optimized for mobile, tablet, and desktop views |
| Device Preview | Simulate iPhone SE, iPhone 14 Pro, and iPad dimensions |
| Interactive Navigation | Collapsible hamburger menu on mobile devices |
| Custom SVG Avatar | Hand-crafted floral-themed profile illustration |
| Toast Notifications | User feedback for device preview actions |
| Gradient Styling | Modern CSS gradients and backdrop filters |

## Technologies

- HTML
- CSS (Flexbox, Grid, CSS Variables, Transitions)
- SVG Graphics
